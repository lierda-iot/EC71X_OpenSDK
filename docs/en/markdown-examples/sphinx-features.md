To enable it: 1. Install: `pip install sphinx-tabs` 2. Add to `extensions` in `conf_common.py`: `'sphinx_tabs.tabs'` Example syntax: `````markdown ````{tabs} ```{group-tab} Python ```python def hello(): print("Hello from Python!") ``` ``` ```{group-tab} Java Script ```javascript function hello(){ console.log("Hello from Java Script!"); } ``` ``` ```` ````` ## Custom Directives ### Raw HTML ```{raw} html <div style="background-color: #f0f0f0; padding: 10px; border-radius: 5px;"> <p>This is custom HTML content.</p> </div> ``` ### Container with Custom Classes ```{container} custom-class This content is wrapped in a div with class "custom-class". ``` ## Index Entries {index}`single: Python; examples` This creates an index entry for "Python; examples". ## Footnotes Here is a footnote reference [^1].
