Switch Language
=========================

:link_to_translation:`zh_CN:[中文]`

This documentation is available in two languages. In case of any discrepancy, the English version shall prevail.

- English Version
- Chinese Version

You can easily switch languages by clicking the language link at the top of each document.

.. image:: ../_static/choose_language.png

Steps to switch language:

1. Look for the language link at the top of any page (e.g., ``:link_to_translation:`zh_CN:[中文]``)
2. Click the link to switch to the translated version
3. The URL will automatically update to reflect the new language

.. note::

   Not all pages have translations. If no translation is available, you will be redirected to the English version.
