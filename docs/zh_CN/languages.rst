切换语言
=========================

:link_to_translation:`en:[English]`

本档现提供两种语言版本。如有任何差异，请以英文版为准。

- 英文版
- 中文版

您可以通过点击每个文档顶部语言链接轻松切换语言。

.. image:: ../_static/choose_language.png

切换语言步骤：

1. 在任何页面顶部查找语言链接（例如：``:link_to_translation:`en:[English]``）
2. 点击链接即可切换到翻译版本
3. URL 将自动更新以反映新语言

.. note::

    并非所有页面都有翻译。如果没有可用的翻译，您将被重定向到英文版本。
